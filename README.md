# Week 1 - Python Client Project

## Project Title:
Basic Data Processing Script - Average Temperature Calculator

## Description:
This project is a simple Python program that:
- Takes temperature inputs from the user
- Stores them in a list
- Calculates total temperature
- Computes average temperature

## Technologies Used:
- Python

## How to Run:
1. Install Python
2. Run:
   python average_temperature.py

## Learning Outcomes:
- Python variables and data types
- Input and Output operations
- Loops (for loop)
- Basic data processing
